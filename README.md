# 📊 Employee Behavior Under Performance Pressure

## 🚀 Overview
This project analyzes how high performance expectations influence unethical behavior among employees through psychological mechanisms.

---

## 🎯 Objective
To explore how performance pressure impacts employee behavior using a "double-edged sword" model:
- Threat perception → increases unethical behavior
- Challenge perception → reduces unethical behavior

---

## 🔬 Hypotheses
- H1: High expectations increase unethical behavior through threat appraisal
- H2: High expectations decrease unethical behavior through challenge appraisal
- H3: Resilience moderates both relationships

---

## 🧩 Dataset
- 79 employees
- 10 days tracking
- Features:
  - Performance Expectation
  - Threat Score
  - Challenge Score
  - UPB Score
  - Resilience Score

---

## 📊 Dashboard
Interactive dashboard built using Python (Plotly) to visualize:
- Expectation vs Threat
- Expectation vs Challenge
- Threat vs UPB
- Challenge vs UPB

---

## 💡 Key Insights
- Performance expectations do not directly cause unethical behavior
- Employee perception is the key driver
- Low resilience increases ethical risk

---

## 💼 Business Value
- Helps organizations reduce ethical risks
- Improves employee experience
- Supports better performance management strategies

---
